Corpuri geometrice

1) Prisma:

Elementele prismei:
- lungimea (AB = L);
- lățimea (BC = l);
- diagonala (A'C = d);
- înălțime (AA'= h).

2) Piramida:

Elementele Piramidei:
- muchii laterale (VA, VB, VC);
- apotema bazei(OM = A.b.);
- apotema piramidei(VM = A.p.);
- înălțime (VO = h).

3) Conul:

Elementele conului:
- raza (AO = OB = R);
- generatoarea (VA = VB = G);
- înălțimea (VO = h);
- diametru(AB = D);
