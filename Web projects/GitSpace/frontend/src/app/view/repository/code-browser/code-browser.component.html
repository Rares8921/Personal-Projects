Quick setup — using your Custom Git CLI

Option 1: Create a new repository

# 1. Initialize
java -jar git-cli.jar init

# 2. Create a file
echo "# {{repo.name}}" > README.md

# 3. Snapshot the directory (Copy the output HASH!)
java -jar git-cli.jar write-tree
# Output example: a1b2c3...

# 4. Create Commit (Paste the TREE_HASH from above)
java -jar git-cli.jar commit-tree <TREE_HASH> -m "Initial commit"

# 5. Push (Requires Login first)
java -jar git-cli.jar login
java -jar git-cli.jar push {{repo.cloneUrlHttp}} master

Option 2: Push existing folder

# Assuming you are in the project folder
java -jar git-cli.jar login

# Snapshot and Commit
java -jar git-cli.jar write-tree
java -jar git-cli.jar commit-tree <TREE_HASH> -m "Push existing"

# Push to this remote
java -jar git-cli.jar push {{repo.cloneUrlHttp}} master
