Data Privacy Protocol

Last updated: System Cycle 2026.01

1. Data Collection

We collect essential telemetry data including: username, email address (for authentication tokens), and Git repository metadata. All code pushed to private repositories is encrypted at rest.

2. Source Code Usage

Your source code belongs to you. We do not analyze, sell, or train AI models on your private repositories without explicit consent via the System Settings module.

3. Cookies & Local Storage

We use JWT Tokens stored in LocalStorage for session persistence. No third-party tracking cookies are injected into the neural interface.

4. Account Termination

You may request a full data purge via the Settings panel. This action is irreversible and will wipe all repositories and commit history.
